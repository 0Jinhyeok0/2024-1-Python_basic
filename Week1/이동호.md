# Problem_Title

 - 문제 링크
  
화씨와 섭씨 온도간의 변환
1. 섭씨 30℃는 화씨로 ℉인지 ?
2. 화씨 150℉는 섭씨로 ℃인지 ?
### 풀이 과정  



### 소스 코드

```Language
c=30
f=c*(9/5)+32
print('섭씨{}℃ = 화씨{}℉'.format(c,f))

f=150
c=(f-32)*(5/9)
print('화씨{}℉ = 섭씨{}℃'.format(f,c))
```

### 결과 화면
섭씨30℃ = 화씨86.0℉
화씨150℉ = 섭씨65.55555555555556℃
