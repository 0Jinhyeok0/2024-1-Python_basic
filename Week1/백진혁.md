# Problem_Title

 - 문제 링크
  
  화씨와 섭씨 온도간의 변환

1. 섭씨 30℃는 화씨로 ℉인지 ?
2. 화씨 150℉는 섭씨로 ℃인지 ?

### 풀이 과정  



### 소스 코드

```Language
c = 30
f = c * 9/5 +32
print('섭씨', c,'℃ 는 화씨', f, '℉ 이다.')

f = 150
c = (f-32)*5/9
print('화씨', f,'℉ 는 섭씨', c, '℃ 이다.')
```

### 결과 화면
섭씨 30 ℃ 는 화씨 86.0 ℉ 이다.
화씨 150 ℉ 는 섭씨 65.55555555555556 ℃ 이다.
